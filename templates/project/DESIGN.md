## Component Patterns

### Buttons
- **Primary:** Solid background, white text, hover darken 10%
- **Secondary:** Outline, hover fill
- **Ghost:** No border, hover background
- **Size:** `sm` (32px), `md` (40px), `lg` (48px)

### Cards
- Background: surface color
- Border: 1px subtle or shadow
- Radius: `md`
- Padding: `lg`

### Forms
- Input height: 40px (md), 48px (lg)
- Label: above, font-medium
- Error: red-500, below input
- Focus ring: 2px primary

## Accessibility

- **Contrast:** WCAG AA minimum (4.5:1 text, 3:1 large text)
- **Focus:** Visible focus rings on all interactive elements
- **Motion:** Respect `prefers-reduced-motion`
- **Touch targets:** Minimum 44x44px on mobile
- **Semantic HTML:** Use correct heading hierarchy, ARIA where needed
